# StarLib ⭐

Applicazione desktop per la gestione di librerie virtuali, prodotta con C++ e Qt.

<img src="https://raw.githubusercontent.com/elia-stellin-unipd/StarLib/refs/heads/main/.relazione/schermata_principale.png">


Per una descrizione più approfondita delle funzionalità e del design dell'applicativo è possibile consultare la [relazione annessa](./.relazione/Relazione.pdf).


## Funzionalità principali
1. **Gestione di tre tipi di articoli** (libri, film, riviste di giornale):
    - Aggiunta
    - Modifica
    - Rimozione dalla libreria 
    - Possibilità di aggiungere articoli tra i "preferiti"

1. **Gestione delle librerie**:
    - Creazione di nuove librerie
    - Possibilità di aprire file di libreria, trascinando un file JSON valido oppure aprendolo usando il menù delle azioni
    - Modifica dei dati della libreria

1. **Ricerca di articoli all'interno di una libreria**
    - Ricerca testuale parziale e case-insensitive
    - Ricerca avanzata per tipo
    - Ricerca tra gli elementi preferiti

1. **Conversione e salvataggio in formato JSON**
